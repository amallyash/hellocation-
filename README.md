# Hellocation 🇰🇷

**Hellocation** adalah tempat saya menjalani internship sebagai **UI/UX Designer**. Repository ini berisi hasil pengembangan landing page website Hellocation — sebuah platform yang membantu para pelajar mempersiapkan diri kuliah ke luar negeri, mulai dari informasi beasiswa, mentorship, legalisasi dokumen, hingga persiapan bahasa.

## 📌 Tentang Proyek

Selama internship, saya berkesempatan untuk mendesain ulang tampilan UI/UX Hellocation sekaligus mengembangkan implementasinya ke dalam bentuk website menggunakan HTML & CSS. Proyek ini menjadi media belajar sekaligus portofolio saya dalam menerapkan desain ke kode nyata.

## 🎯 Tujuan

- Mendesain ulang antarmuka (UI) Hellocation agar lebih menarik dan mudah digunakan (UX)
- Mengimplementasikan desain ke dalam website yang responsif
- Terus mengembangkan fitur dan halaman baru seiring berjalannya waktu

## 🛠️ Teknologi yang Digunakan

- HTML5
- CSS3
- (akan ditambahkan seiring pengembangan lebih lanjut, misalnya JavaScript)
